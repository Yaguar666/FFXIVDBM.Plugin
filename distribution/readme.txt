Welcome to Deadly Boss Mods - FFXIV edition!

Make sure you have a chat tab with ALL filters enabled, or the plugin won't work right!

Encounters currently supported:
* Titan Hard Mode (was my test encounter when first coding the plugin)
* Binding Coil Turns 6-13 (SCOB 1-4, FCOB 1-4)
* Ravana Extreme
* Alexander Savage Floor 1

The encounters are only working for the English client at the moment. I will need assistance from users of other language clients to get it working for them.
